1.) Compile the java file "InternetChatting.java" using the command "javac InternetChatting.java"
2.) Run the program using "java InternetChatting.java" command
3.) It will show the port number and prompts to enter the other side's port number -
	Enter it in the following format - "port_number your_name"
	For Example, if your name is Bob and the port number you want to connect to is 1234, enter "1234 Bob"
4.) If you want to write a message, type it and press enter.
5.) If you want to transfer a file, type "transfer filename" and press enter.
